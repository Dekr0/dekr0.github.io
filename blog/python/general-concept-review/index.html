set类型

与list和tuple（不允许修改的列表），set是一种不允许重复的列表并且是无序的，所以不支持索引

类set的方法（不存在于list和tuple）：

  • add(self, *args, **kwargs) - 添加一个元素。如果元素存在，则无效果

  • differece(self, *args, **kwargs) - 传入一个或多个set对象，如果调用此方法set存在的元素不存在于被传入的set，将这些元素以一个新的set对象返回

  • difference_update(self, *args, **kwargs) - 传入一个set对象，如果调用此方法的set对象存在的元素不存在于被传入的set，保留这些元素并移除其他存在于传入的set对象的元素

  • discrad(self, *args, **kwargs) - 移除一个是此对象成员的元素，如果元素不是对象的成员，不执行（与remove功能相同，但不会因为元素不存在而报错）

  • intersection(self, *args, **kwargs) - 传入一个或多个set对象，如果调用此方法的set对象存在的元素同样存在于被传入的set，将这些元素以一个新的set对象返回

  • intersection_update(self, *args, **kwargs) - 传入一个set对象，如果调用此方法的set对象存在的元素同样存在于被传入的set，保留这些元素并移除其他不在于传入的set对象的元素

  • isdisjoint(self, *args, **kwargs) - 如果两个set对象没有相交，返回True

  • issubset(self, *args, **kwargs) - 如果此set对象是另一个set对象的子集合，返回True

  • issuperset(self, *args, **kwargs) - 如果此set对象是另一个set对象的父集合，返回True

  • symmetric_difference(self, *args, **kwargs) - 返回两个或者多个set对象中互不存在的元素

  • symmetric_difference_update(self, *args, **kwargs) - 保留两个set对象中互不存在的元素，其他的移除

  • union(self, *args, **kwargs) - 返回两个set对象的并集

  • update(self, *args, **kwargs) - 传入一个可循环的对象（列表，元组等），并更新与这对象的并集

Remark：pop方法无法传参，被移除的元素是随机的

三元计算

简化一些简单if-else statement

var = value_1 if condition else value_2

如果conidtion成立，取value_1，否则取value_2

Remark：逻辑判断同样可以利用在赋值和其他应用中

obj = (class_A or class_B)()  # class_B does not exit
print(obj)

# output: an index / memory address of a class_A's instance

lambda表达式

简化一些简单的函数体

function_name = lambda arguments : …

addition = lambda x, y: x + y
result = addition(1,2)

def addition(x, y, i=0):
    if i == 10:
        return x
    z = x + y
    i += 1
    addition(y, z, i)


result = addition(0, 1)
# result is None

由于调用函数自身的时候，没有return来接收底层函数的返回值，只是执行函数体本身，直到最外层的函数也没有接收任何的返回值。在返回到最外层的时候，由于函数体没有设置关键字return，函数本身将不会返回任何值，变量则等于None。

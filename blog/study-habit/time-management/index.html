项目和考试复习规划 (Monthly Schedules)

  • 尽量提前且经常复习, 减小为 exam 前做的准备且提升知识弹性

  • 提前设立 exam 复习时间表, 留有充足的预算时间, 提前 3 至 4 个星期开始 exam 复习或一个项目, 在考试日或项目 deadline 前几日前完成复习 / 项目

    • 设立开始日期
    • 罗列需要完成的主要任务, 并解分成小步骤 / sections
    • 将每一项任务分配到一个特定的星期
    • 在每个星期的课程学习时间段, 着手完成这些分配的任务

  • 使用 back-planning 的方法来进行规划, 即从考试日 / 项目 deadline 开始往后进行规划

每周时间表 / 日程安排 (Weekly Schedules)

可分为几个块:

  • Class Time - 标注为上课, 做 labs 或 seminars, etc.

  • Personal Time - 标注为paid work, 额外 curricular 活动, 健身, etc.

  • Study Time - 在一个星期为每一节课程预留至少 2 个小时进行学习 (复习)

    • 如在 24 小时内完成作业 / 任务, 阅读, 回顾笔记
    • 不要将学习 seesion 的时间定制的太长 (5 one hour study sessions are better than 1 five-hour session)
    • 注意休息 (由此在远程教学这段时间)
    • 利用较短的时间段来做一些特定 / 详细的任务

  • Assignment Time - 进行 research, 写 papers, 完成日常的和一次性的作业任务

    • 为日常作业任务做天数/时间规划

  • Flex Time - 可变动, 为每一日 / 星期预留小部分这些的时间来减少压力, keep things manageable

在制定每周日程安排的时候, 尽量没要出现灰色时间段, 即没分配大概 / 特定活动的时间段

每周日程安排与每月的安排相配合

综合来看, 将每一个星期分解成多个可替换, 交换的时间 / 非详细活动块, 以应变在预期之外的事情

分配时间要基于自身能力和真实情况, 不要 over-planning (firm but flexible), 或制作出来的计划超出了自身的执行能力, 即无法长期保持每周时间表的执行

Daily to-dos

此规划更适用于计划事项, 如考试复习安排, 一次性作业任务, etc. - Daily to-dos 可以是 appointments, 日常任务, etc. - 配合每周和每月的日程安排 - 提前设下 reminders

时间表 / 日程安排的执行

保持一个稳定的学习过程循环:

  • 预览 (Preview)
    • 在学习或复习前, 提前预览并首先找出重要或具有类似特点的内容
  • 学习 (Learn)
  • 回忆 (Retrieve) <-> 复习 (Review),
    • 在复习前, 先尝试回忆之前学过的内容, 之后在对照正确答案进行核实
    • 在复习后, 在进行回忆

时刻留意制定的日程安排是否有效, 是否按时执行, 并对日程安排中不足的地方进行调整

  • 很多时候日程安排需要不断去做出尝试, 调整纠正, 复查
